CC Code Camp

Want to learn to code?

This summer camp is meant to be a fun easy introduction to coding for students in Bend, Oregon. Throughout the four days of camp, we will be using multiple coding apps and educational tools that will teach kids the fundamentals of coding. All of the necessary technology will be provided. Camps take place from Monday through Thursday. Half way through each camp there will be an included snack. If you have any other questions about the schedule, please don't hesitate to email us at 1chasemcd@gmail.com.

Programs

Intro to Programming

In this program, Bend, Oregon students grades 2 to 5 will gain basic knowledge on coding. We will start by using a simple language called Scratch to learn about the basic tools of coding. Scratch uses colorful blocks of code that can be used together to create games. This will teach the concepts involved in all programing languages, while avoiding the complex syntax used in written programming languages like Java and Python. After kids learn the basics of programming, they will use their new skills to program robots such as spheros, Lego Mindstorms robots, and more!

Code Camp 2

In our second offered program, Bend, Oregon students grades 3 to 6 will have the opportunity to build off of what they learned in the previous program. We will start by reviewing what was learned in the introductory course, then jump right back into programming with the language Scratch. This time, kids will learn more advanced skill to use when making fun games. All of the skills learned using this language will then be applied to programming robots, such as Lego Mindstorms and spheros, which both provide a fun way to use computer skills. This more advanced camp will be a blast for kids that love to code!
